"day 1: learned some more commands"
""
"use vim filename.txt to enter vim to edit text files"
"press i to change into insert mode to write to a text file"
"when you are done making changes, press esc to exit insert mode"
"then type :wq to write and quit"
""
""
"git init : make a new repo in the current directory"
"git status : show staged files, untracked files, and anything else that is about the state of the current repository"
"git add : use this to add files to stage them to be pushed to Github"
"example to add all files : git add ."
"then after using git add ., you use git commit to commit changes."
"usually one uses: git commit -m "example message" to commit changes with a message about what the changes were."
""
""
"git log can be used to see the list of all changes made to a repository"
"you can use: git checkout to switch branches to restore working tree files"
"example: git checkout --<file>   can be used to discard changes in the working directory"
"can use git branch <branch-name> to create a new branch so that you can make changes to the code without messing up the main branch"
"git checkout <branch-name> is used to switch between branches"
"eventually you will need to merge branches to something like the main branch"
"you can do that by using the command: git merge <branch-name> to merge the named branch to the current branch"
""
""
"git remote lets you create, view and delete connections to other repositories"
"use git remote add <name> <url> to connect to a new remote repo. make a name in the <name> section to refer to the remote repo"
"git fetch: downloads changes and files to your local repo"
"git pull fetches and merges changes from remote repos to your current branch"
